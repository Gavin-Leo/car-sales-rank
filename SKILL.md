# 懂车帝汽车销量排行榜

从懂车帝公开 API 获取月度汽车零售销量数据，生成专业 Excel 报表。

## 工作流程

### 1. 确定目标月份

- 用户未指定月份时，使用当前月份的上一个月（如当前为 2026 年 6 月，则查询 202605）
- 用户指定月份时，格式为 YYYYMM（如 `202604` 表示 2026 年 4 月）
- 用户指定"最新"、"最近一个月"时，使用上个月

### 2. 执行脚本生成 Excel

运行 `scripts/generate_excel.py`，传入月份和输出路径：

```bash
python scripts/generate_excel.py --month YYYYMM --count 100 --output <输出路径>
```

参数说明：
- `--month` / `-m`：(必填) 月份，格式 YYYYMM
- `--count` / `-c`：(可选) 获取车型数量，默认 100
- `--output` / `-o`：(必填) 输出的 .xlsx 文件绝对路径

脚本自动完成：
1. 调用懂车帝 API 获取车型销量数据
2. 从车型数据聚合计算品牌总销量
3. 生成包含两个 Sheet 的 Excel 文件

### 3. 结果呈现

生成成功后，必须向用户展示：
- Excel 文件路径
- 车型销量榜前 10 名摘要（排名、车型、品牌、销量）
- 品牌销量榜前 5 名摘要（排名、品牌、总销量）

使用表格形式呈现摘要数据，然后附带 Excel 文件路径。

## 生成的 Excel 结构

### Sheet 1：车型销量榜

| 列 | 字段 | 说明 |
|----|------|------|
| A | 排名 | 当前月销量排名 |
| B | 排名变化 | 较上月变化（正=上升，负=下降） |
| C | 车型名称 | 车系名称 |
| D | 品牌 | 所属品牌 |
| E | 指导价（万） | 价格区间 |
| F | 销量（辆） | 当月零售销量 |
| G | 点评数 | 懂车帝点评数 |

特性：
- 自动筛选（AutoFilter）
- 冻结首行
- 排名变化列条件着色（绿=上升，红=下降）
- 隔行交替背景色
- 销量数字千分位格式

### Sheet 2：品牌销量榜

| 列 | 字段 | 说明 |
|----|------|------|
| A | 排名 | 按总销量降序 |
| B | 品牌名称 | 品牌名 |
| C | 总销量（辆） | 旗下所有车型销量之和 |
| D | 旗下车型数 | 上榜车型数量 |
| E | 均价区间（万） | 旗下车型均价范围 |
| F | 市场份额 | 占上榜总销量百分比 |

特性：
- 自动筛选（AutoFilter）
- 冻结首行
- 底部数据来源说明行

## 依赖

脚本自动安装缺失的依赖（`requests`、`openpyxl`），无需手动安装。

## 注意事项

- 懂车帝 API 为公开接口，无需认证
- 数据为零售销量，非批发/上险数据
- 单次请求建议 count ≤ 100，脚本自动分页获取
- 如 API 请求超时，可重试；连续失败应告知用户稍后再试
