# car-sales-rank

国内汽车销量排行榜查询与 Excel 生成工具

## 功能

从懂车帝公开 API 获取月度汽车零售销量数据，自动生成带筛选和条件格式的专业 Excel 报表，包含两个 Sheet：

- **车型销量榜**：排名、排名变化、车型名称、品牌、指导价、销量、点评数
- **品牌销量榜**：排名、品牌名称、总销量、车型数、均价区间、市场份额

## 触发词

"汽车销量""车型销量榜""品牌销量榜""销量排行""销量排名""买车排行榜""懂车帝销量""月销量""卖了多少辆""哪款车卖得好""汽车销量 Excel""销量表"

## 使用方式

在马维斯中说出触发词即可，例如：

> "帮我查一下上个月的汽车销量排行"

支持指定月份（格式 YYYYMM），不指定时自动使用最新可用月份。

## 目录结构

```
car-sales-rank/
├── SKILL.md                  # 技能定义文件
├── scripts/
│   └── generate_excel.py     # 销量数据获取与 Excel 生成脚本

```


## Excel 特性

- 自动筛选（AutoFilter）
- 冻结首行
- 排名变化条件着色（绿升红降）
- 隔行交替背景色
- 销量千分位格式

## 数据来源

懂车帝公开 API，无需认证。数据为零售销量口径。

## 依赖

脚本运行时自动安装 `requests` 和 `openpyxl`，无需手动干预。
